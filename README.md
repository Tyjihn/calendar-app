# CalendAIr

CalendAIr is an early-stage AI-powered study and planning assistant designed to help students organize tasks, build daily study plans, and improve productivity.

## Current Stage
MVP Phase 1: UI foundation (dashboard + task layout)

## Tech Stack
- Next.js
- React
- (Tailwind if you're using it)

## Features (in progress)
- Dashboard layout
- Task display system
- Study planning interface (mock data)

## Roadmap
- Task creation system
- Persistent storage
- AI-based scheduling assistant
